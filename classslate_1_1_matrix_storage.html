◆ release()

template<typename scalar_t >
void slate::MatrixStorage< scalar_t >::release ( ijdev_tuple  ijdev)

Remove a tile instance on device and delete it if it is a workspace and not OnHold nor Modified.

If tile node becomes empty, deletes it. If tile's memory was allocated by SLATE, then its memory is freed back to the allocator memory pool. device can be AllDevices.

Note that Modified, local tiles are currently not released but that this behavior may change in the future and should not be relied apon.
