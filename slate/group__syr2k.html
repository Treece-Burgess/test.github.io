Distributed parallel symmetric rank 2k update.

Performs the symmetric rank 2k operation

\[ C = \alpha A B^T + \alpha B A^T + \beta C, \]

where alpha and beta are scalars, C is an n-by-n symmetric matrix, and A and B are an n-by-k matrices. The matrices can be transposed beforehand, e.g., 

auto AT = slate::transpose( A );
auto BT = slate::transpose( B );
slate::syr2k( alpha, AT, BT, beta, C );
Template Parameters
scalar_tOne of float, double, std::complex<float>, std::complex<double>.
Parameters
[in]alphaThe scalar alpha.
[in]AThe n-by-k matrix A.
[in]BThe n-by-k matrix B.
[in]betaThe scalar beta.
[in,out]COn entry, the n-by-n symmetric matrix C. On exit, overwritten by the result \(C = \alpha A B^T + \alpha B A^T + \beta C\).
[in]optsAdditional options, as map of name = value pairs. Possible options:
  • Option::Lookahead: Number of blocks to overlap communication and computation. lookahead >= 0. Default 1.
  • Option::Target: Implementation to target. Possible values:
    • HostTask: OpenMP tasks on CPU host [default].
    • HostNest: nested OpenMP parallel for loop on CPU host.
    • HostBatch: batched BLAS on CPU host.
    • Devices: batched BLAS on GPU device.
